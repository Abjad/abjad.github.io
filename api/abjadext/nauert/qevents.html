Pitched q-event.

Indicates the onset of a period of pitched material in a q-event sequence.

>>> pitches = [0, 1, 4]
>>> q_event = abjadext.nauert.PitchedQEvent(1000, pitches)
>>> abjad.f(q_event)
abjadext.qevents.PitchedQEvent(
    offset=abjad.Offset((1000, 1)),
    pitches=(
        abjad.NamedPitch("c'"),
        abjad.NamedPitch("cs'"),
        abjad.NamedPitch("e'"),
        ),
    )

class abjadext.nauert.qevents.QEvent(offset=0, index=None)[source]Â¶

Abstract Q-event.

Represents an attack point to be quantized.

All QEvents possess a rational offset in milliseconds, and an optional index for disambiguating events which fall on the same offset in a QGrid.
