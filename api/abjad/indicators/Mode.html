Mode¶

class abjad.indicators.Mode.Mode(mode_name='dorian')¶

Mode.

Initializes from string:

>>> abjad.Mode("major")
Mode('major')

Initializes from other mode:

>>> mode = abjad.Mode("dorian")
>>> abjad.Mode(mode)
Mode('dorian')

overridden __eq__(argument)¶

Is true when argument is a mode with mode name equal to that of this mode.

>>> mode_1 = abjad.Mode("major")
>>> mode_2 = abjad.Mode("major")
>>> mode_3 = abjad.Mode("dorian")

>>> mode_1 == mode_1
True

>>> mode_1 == mode_2
True

>>> mode_1 == mode_3
False

>>> mode_2 == mode_1
True

>>> mode_2 == mode_2
True

>>> mode_2 == mode_3
False

>>> mode_3 == mode_1
False

>>> mode_3 == mode_2
False

>>> mode_3 == mode_3
True

Returns true or false.

named_interval_segment¶

Gets named interval segment.
