RehearsalMark¶

class abjad.indicators.RehearsalMark.RehearsalMark(*, markup=None, number=None, tweaks=None)¶

Rehearsal mark.

>>> staff = abjad.Staff("c'4 d' e' f'")
>>> score = abjad.Score([staff])
>>> mark = abjad.RehearsalMark(number=1)
>>> abjad.attach(mark, staff[0])
>>> scheme = abjad.Scheme("format-mark-box-alphabet")
>>> abjad.setting(score).markFormatter = scheme
>>> abjad.show(score)

>>> abjad.show(staff)
