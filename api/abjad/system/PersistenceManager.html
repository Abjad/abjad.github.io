PersistenceManager¶

class abjad.system.PersistenceManager.PersistenceManager(client=None)¶

Persistence manager.

>>> staff = abjad.Staff("c'4 e'4 d'4 f'4")
>>> abjad.show(staff)  

>>> abjad.persist(staff)
PersistenceManager(client=Staff("c'4 e'4 d'4 f'4"))

as_png(flags='--png', illustrate_function=None, png_file_path=None, preview=False, remove_ly=False, resolution=False, **keywords)¶

Persists client as PNG.

>>> staff = abjad.Staff()
>>> container = abjad.Container("c'4 d'4 e'4 f'4")
>>> command = abjad.LilyPondLiteral(r"reak", "after")
>>> abjad.attach(command, container[-1])
>>> staff.extend(200 * container)

>>> result = abjad.persist(staff).as_png()  
>>> for x in result[0]:  
...     x
... 
'/Users/josiah/.abjad/output/0110-page1.png'
'/Users/josiah/.abjad/output/0110-page2.png'
'/Users/josiah/.abjad/output/0110-page3.png'
