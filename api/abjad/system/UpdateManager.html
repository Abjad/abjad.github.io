UpdateManager¶

class abjad.system.UpdateManager.UpdateManager¶

Update manager.

Updates start offsets, stop offsets and indicators everywhere in score.

Note

This is probably the most important part of Abjad to optimize. Use the profiler to figure out how many unnecessary updates are happening. Then reimplement. As a hint, the update manager implements a weird version of the “observer pattern.” It may make sense to revisit a textbook example of the observer pattern and review the implementation of the update manager.
