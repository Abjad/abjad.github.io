TremoloContainer¶

class abjad.core.TremoloContainer.TremoloContainer(count=2, components=None, *, tag=None)¶

Tremolo container.

>>> staff = abjad.Staff()
>>> staff.append(abjad.TremoloContainer(2, "c'16 e'"))
>>> staff.append("cs'4")
>>> staff.append(abjad.TremoloContainer(2, "d'16 f'"))
>>> staff.append("ds'4")
>>> abjad.show(staff)

Duration of container equal to contents duration multiplied by count:

>>> abjad.inspect(staff[0]).duration()
Duration(1, 4)

Duration of each leaf equal to written duration multiplied by count:

>>> abjad.inspect(staff[0][0]).duration()
Duration(1, 8)

(Container).__delitem__(i)¶

Deletes components(s) at index i in container.

Deletes first tuplet in voice:

>>> voice = abjad.Voice()
>>> voice.append(abjad.Tuplet((2, 3), "c'4 d'4 e'4"))
>>> voice.append(abjad.Tuplet((2, 3), "e'4 d'4 c'4"))
>>> leaves = abjad.select(voice).leaves()
>>> abjad.slur(leaves)
>>> abjad.show(voice)  

>>> tuplet_1 = voice[0]
>>> del voice[0]
>>> start_slur = abjad.StartSlur()
>>> leaf = abjad.select(voice).leaf(0)
>>> abjad.attach(start_slur, leaf)

First tuplet no longer appears in voice:

>>> abjad.show(voice)  

>>> abjad.inspect(voice).wellformed()
True

First tuplet must have start slur removed:

>>> abjad.detach(abjad.StartSlur, tuplet_1[0])
(StartSlur(),)

>>> abjad.show(tuplet_1)  

>>> abjad.inspect(tuplet_1).wellformed()
True

Returns none.
