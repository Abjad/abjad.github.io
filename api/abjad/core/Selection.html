Selection¶

class abjad.core.Selection.Selection(items=None, previous=None)¶

Selection of items (components / or other selections).

>>> string = r"c'4 \times 2/3 { d'8 r8 e'8 } r16 f'16 g'8 a'4"

>>> abjad.setting(staff).auto_beaming = False
>>> abjad.show(staff)  

>>> result = abjad.select(staff).runs()

>>> for item in result:
...     item
... 
Selection([Note("c'4"), Note("d'8")])
Selection([Note("e'8")])
Selection([Note("f'16"), Note("g'8"), Note("a'4")])

__illustrate__()¶

Attempts to illustrate selection.

Evaluates the storage format of the selection (to sever any references to the source score from which the selection was taken). Then tries to wrap the result in a staff; in the case that notes of only C4 are found then sets the staff context name to 'RhythmicStaff'. If this works then the staff is wrapped in a LilyPond file and the file is returned. If this doesn’t work then the method raises an exception.

The idea is that the illustration should work for simple selections of that represent an essentially contiguous snippet of a single voice of music.

Returns LilyPond file.

Is true when items in selection are contiguous components in the same logical voice.

>>> staff = abjad.Staff("c'4 d'4 e'4 f'4")
>>> staff[:].are_contiguous_logical_voice()
True

>>> selection = staff[:1] + staff[-1:]
>>> selection.are_contiguous_logical_voice()
False

REGRESSION. Before-grace music may be ignored:

>>> voice = abjad.Voice("c'4 d' e' f'")
>>> container = abjad.BeforeGraceContainer("cs'16")
>>> abjad.attach(container, voice[1])
>>> abjad.show(voice)  

>>> voice[:]

>>> voice[:].are_contiguous_logical_voice()

>>> voice[:].are_contiguous_logical_voice(ignore_before_after_grace=True)
