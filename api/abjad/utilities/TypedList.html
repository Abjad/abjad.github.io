TypedList¶

class abjad.utilities.TypedList.TypedList(items=None, item_class=None, keep_sorted=False)¶

Typed list.

No item coercion:

>>> list_ = abjad.TypedList()
>>> list_.append(23)
>>> list_.append("foo")
>>> list_.append(False)
>>> list_.append((1, 2, 3))
>>> list_.append(3.14159)

>>> abjad.f(list_)
abjad.TypedList(
    [
        23,
        'foo',
        False,
        (1, 2, 3),
        3.14159,
        ]
    )

Named pitch item coercion:

>>> pitch_list = abjad.TypedList(item_class=abjad.NamedPitch,)
>>> pitch_list.append(0)
>>> pitch_list.append("d'")
>>> pitch_list.append(("e", 4))
>>> pitch_list.append(abjad.NamedPitch("f'"))

>>> abjad.f(pitch_list)
abjad.TypedList(
    [
        abjad.NamedPitch("c'"),
        abjad.NamedPitch("d'"),
        abjad.NamedPitch("e'"),
        abjad.NamedPitch("f'"),
        ],
    item_class=abjad.NamedPitch,
    )

Ordered collection with optional item coercion.

Implements the list interface.
