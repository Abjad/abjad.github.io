LengthInequality¶

class abjad.utilities.LengthInequality.LengthInequality(operator_string='<', length=None)¶

Length inequality.

>>> inequality = abjad.LengthInequality("<", 4)
>>> abjad.f(inequality)
abjad.LengthInequality(
    operator_string='<',
    length=4,
    )

>>> inequality([1, 2, 3])
True

>>> inequality([1, 2, 3, 4])
False

>>> inequality([1, 2, 3, 4, 5])
False
