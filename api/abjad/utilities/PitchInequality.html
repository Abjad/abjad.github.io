PitchInequality¶

class abjad.utilities.PitchInequality.PitchInequality(operator_string='&', pitches=None)¶

Pitch inequality.

>>> inequality = abjad.PitchInequality("&", "C4 E4")
>>> abjad.f(inequality)
abjad.PitchInequality(
    operator_string='&',
    pitches=abjad.PitchSet(
        [0, 4]
        ),
    )

>>> inequality(abjad.Staff("d'8 e' f' g'"))
True

>>> inequality(abjad.Staff("e'8 f' g' a'"))
True

>>> inequality(abjad.Staff("f'8 g' a' b'"))
False

Note

only intersection currently implemented.
