PartManifest¶

class abjad.segments.PartManifest.PartManifest(*arguments)¶

Part manifest.

Initializes from parts:

>>> part_manifest = abjad.PartManifest(
...     abjad.Part(section="BassClarinet", section_abbreviation="BCL"),
...     abjad.Part(section="Violin", section_abbreviation="VN"),
...     abjad.Part(section="Viola", section_abbreviation="VA"),
...     abjad.Part(section="Cello", section_abbreviation="VC"),
... )
>>> len(part_manifest)
4

Initializes from orchestra sections:

>>> part_manifest = abjad.PartManifest(
...     abjad.Section(abbreviation="FL", count=4, name="Flute",),
...     abjad.Section(abbreviation="OB", count=3, name="Oboe",),
...     abjad.Part(section_abbreviation="EH", section="EnglishHorn",),
...     abjad.Section(
...         abbreviation="VN-1", count=18, instrument="Violin", name="FirstViolin",
...     ),
...     abjad.Section(
...         abbreviation="VN-2", count=18, instrument="Violin", name="SecondViolin",
...     ),
... )
>>> len(part_manifest)
44
